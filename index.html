Backdoor-Attacks

Backdoor detector for BadNets trained on the YouTube Face dataset using the pruning defense.

Accomplishments
  • Tools: TensorFlow, tqdm, Python, NumPy, Colab
  • Project Description: In this project, a backdoor detector for BadNets trained on the YouTube Face dataset is designed using the pruning defense technique. The detector takes a backdoored neural network classifier (BadNet) and a validation dataset of clean, labeled images as input. By pruning the last pooling layer of the BadNet based on decreasing average activation values over the validation set, channels are removed until the validation accuracy drops by a specified percentage. The resulting repaired network (G) can correctly classify clean inputs and detect backdoored inputs with N+1 classes.
  • Experimental results on pruned versions of neural networks from the 2020 hacks git repository reveal the impact of pruning on performance metrics. The test accuracies of pruned bad networks on clean datasets varied, with significant observations plotted against attack success rates for the validation dataset. Repairing the bad networks through pruning at rates of 2%, 4%, and 10% resulted in improved performance, with classification accuracy ranging from 84.54% to 95.90%. These findings demonstrate the efficacy of pruning in enhancing both classification accuracy on clean test data and resilience against adversarial attacks.
